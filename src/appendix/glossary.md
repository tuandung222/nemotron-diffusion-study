# Appendix A - Glossary

This glossary collects terms used across the book. Where a term has a precise definition in a specific lecture, the lecture number is shown in square brackets.

---

**Absorbing-state diffusion** - A discrete diffusion process whose forward kernel only allows transitions from any token to a fixed "mask" token. Equivalent to random Bernoulli masking. *[1.2]*

**AR (autoregressive)** - Standard left-to-right language modeling. $p(\mathbf{x}) = \prod_t p(x_t \mid x_{<t})$. *[1.1]*

**Arithmetic intensity (AI)** - FLOPs per byte streamed. Low AI = memory-bound; high AI = compute-bound. *[1.1]*

**Block diffusion** - A factorization $p(\mathbf{x}) = \prod_m p(\mathbf{x}_{(m)} \mid \mathbf{x}_{(<m)})$ where each block is generated by a small diffusion process. *[1.4]*

**Block_size / block_length** - The number of tokens per block in block diffusion. NLD default = 32. *[1.4]*

**Bidirectional attention** - Attention with an unconstrained mask (no causal upper triangle). Used within a diffusion block. *[1.4]*

**Causal attention** - Attention with a lower-triangular mask. Standard for AR. *[1.4]*

**Confidence-threshold sampler** - A diffusion sampler that commits all masked positions whose top-1 probability exceeds $\tau$. NLD default $\tau = 0.9$. *[1.5]*

**D3PM** - Discrete Denoising Diffusion Probabilistic Models. The original framework for discrete-state diffusion (Austin et al., 2021). *[1.2]*

**Diffusion language model (DLM)** - A language model trained with a diffusion (denoising) objective rather than the AR objective. *[1.3]*

**Dual-stream input** - Concatenating a noisy view and a clean view of the same sequence into a single input of length $2L$. Used in NLD training. *[Series 2]*

**Dual-mode** - Inference that can switch between AR and diffusion modes by changing the attention mask. *[Series 2]*

**ELBO** - Evidence Lower BOund. The variational lower bound on $\log p_\theta(\mathbf{x})$ that diffusion training optimizes. *[1.2]*

**Forward process** - In diffusion, the noising process $q(\mathbf{x}_t \mid \mathbf{x}_{t-1})$ that progressively destroys the data. *[1.2]*

**GQA (Grouped-Query Attention)** - Variant of multi-head attention with $n_{kv} < n_h$ key/value heads, used in modern decoder LMs to reduce KV cache memory. *[1.1, prerequisite]*

**KV cache** - The cached key/value tensors from previously processed tokens, used for efficient AR decoding. *[1.1]*

**Linear self-speculation** - NLD's mode that drafts $k$ tokens with the diffusion head and verifies them with the AR head in $O(1)$ extra forwards. *[Series 2]*

**Mask token** - Special vocabulary symbol that represents "this position is unknown". `mask_token_id` in NLD config = 100. *[1.2, 1.3]*

**Masked Diffusion Model (MDM)** - Absorbing-state discrete diffusion with the simplified MDLM loss. *[1.3]*

**MDLM** - Simple and effective Masked Diffusion Language Models (Sahoo et al., 2024). *[1.3]*

**NAT (Non-Autoregressive Translation)** - Predict all output tokens in parallel from the source encoding. Early non-AR baseline. *[1.1]*

**NFE (Number of Function Evaluations)** - Total forward passes to generate a sequence. NFE = sequence_length / TPF. *[1.1]*

**Pixtral encoder** - NVIDIA's vision encoder, used as the vision tower in NLD-VLM-8B. *[Series 3]*

**Quadratic self-speculation** - NLD's variant of self-speculation that uses FlexAttention to do the draft and verify in a single forward. *[Series 2]*

**Remasking** - Allowing a previously-committed token to be re-masked in a later diffusion step. *[1.5]*

**Reverse process** - The learned denoising process $p_\theta(\mathbf{x}_{t-1} \mid \mathbf{x}_t)$ that inverts the forward process. *[1.2]*

**SEDD** - Score Entropy Discrete Diffusion (Lou et al., 2024). *[1.3]*

**Self-speculation** - NLD's flagship inference mode: the diffusion head drafts $k$ tokens and the AR head verifies them, all using shared backbone weights. *[Series 2]*

**SOL (Speed of Light)** - The theoretical maximum TPF achievable by a clairvoyant sampler given a model and a target sequence. *[1.5]*

**TPF (Tokens Per Forward)** - Average number of committed output tokens per model forward pass. AR has TPF = 1. *[1.1]*

**TPOT (Time per Output Token)** - Wall-clock time per generated token after the first. *[1.6]*

**TTFT (Time to First Token)** - Wall-clock time from request arrival to the first output token. *[1.6]*

**$\bar\beta(t)$ / cumulative mask probability** - In absorbing diffusion, the probability that a position is masked at noise level $t$. *[1.2]*

**$\tau$ (threshold)** - In the confidence-threshold sampler, the probability above which a masked position is committed. *[1.5]*

**$1/t$ reweighting** - The schedule-dependent factor that makes the MDLM loss a valid ELBO. *[1.3]*
